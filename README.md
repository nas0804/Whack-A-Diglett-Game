# Whack-A-Diglett-Game
In this repository is a whack a mole game I developed in my spare time using python.
Upon running the program the user will have the option of 3 time durations to play the game for. The objective is to get as many points as possible, Digletts (the Induvidual 'Mole') are worth 1 point, Dugtrios (the 3 'Moles') are worth 3 points and Geodudes (the rock looking sprite) are worth -5 to punish the user.
